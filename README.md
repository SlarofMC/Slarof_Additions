# Slarof Resourcepacks
Дополнительные ресурспаки для Minecraft сервера Slarof:

- [Slarof Additions](https://github.com/SlarofMC/Slarof_Resourcepacks/releases/tag/slarof-additions) - дополнительный контент для сервера
- [КСЭПСП](https://github.com/SlarofMC/Slarof_Resourcepacks/releases/tag/ksepsp) - контент для ролевой игры
  - [КСЕПСП & EcoEnchants](https://github.com/Slarof/RP.KSEPSP-EcoEnchants/releases) - поддержка КСЭПСП при использовании EcoEnchants



### Автоматическая загрузка и обновление текстурпаков
1. Скачайте модификацию [Resclone](https://modrinth.com/mod/resclone/versions) в `.minecraft/mods`
2. Скачайте [конфигурацию](https://www.dropbox.com/scl/fi/jo3jg595ol38jf59u6tuo/config.json?rlkey=capkuedpivhz12kc6wmlxcmor&st=59czwp7g&dl=1) и вставьте в `.minecraft/config`
3. Запустите игру и включите текстурпак в настройках `Наборы ресурсов`
4. Готово
